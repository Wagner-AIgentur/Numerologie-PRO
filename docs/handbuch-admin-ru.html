1. Вход и доступ

Панель администратора доступна по адресу: numerologie-pro.com/ru/admin

Необходимо войти в систему, и ваш профиль должен иметь статус «admin». Если у вас нет доступа, попросите Данила сделать ваш аккаунт администратором.

Совет: Внизу боковой панели есть ссылка «Панель клиента» для переключения на обычный вид.

2. Главная панель (Dashboard)

/admin

Стартовая страница с ключевыми показателями:

  • 4 карточки KPI: Всего клиентов, новых заявок, оплаченных заказов, запланированных сессий
  • Ближайшие сессии: 5 предстоящих встреч с датой, типом и ссылкой на видео
  • Открытые follow-up: CRM-заметки с пометкой «требуется действие» (просроченные выделены красным)
  • Последние заявки: 5 последних сообщений с формы контактов
  • Последние заказы: 5 последних покупок
Совет: Нажмите на карточку KPI, чтобы перейти в соответствующий раздел.

3. Входящие (Inbox)

/admin/inbox

Здесь собраны все действия клиентов — письма, Telegram-сообщения, формы контактов, заказы, сессии, заметки и Instagram-DM.

Вкладки-фильтры

  • Все — Вся активность
  • E-Mail — Отправленные и полученные письма
  • Telegram — Сообщения Telegram
  • Контакты — Записи с формы контактов
  • Заказы — Покупки и сессии
  • Заметки — CRM-заметки и обновления сделок
  • Instagram — Instagram-DM и Lead Ads

Что можно делать?

  • Нажать на запись → Отметить как прочитанную и перейти к профилю клиента
  • «Прочитать все» → Пометить все записи в текущей вкладке как прочитанные
  • «Загрузить ещё» → Подгрузить больше записей

Непрочитанные записи имеют синюю точку и жирный заголовок.

5. Воронка (Kanban-доска)

/admin/pipeline

Визуальная доска с 4 колонками по статусу клиента:

КолонкаЗначение
ЛидНовые потенциальные клиенты, которые ещё не покупали
КлиентКлиенты хотя бы с одним заказом
VIPОсобо важные клиенты

На каждой карточке: имя, email, количество заказов и общая выручка.

Что можно делать?

  • Переместить клиента: Нажмите стрелки на карточке

6. Сделки

/admin/deals

Отслеживайте текущие продажи в 6-этапной воронке:

Новая → Контакт → Предложение → Переговоры → Выиграна → Проиграна

Что показывает каждая сделка?

  • Название и имя клиента
  • Сумма в евро
  • Вероятность (%)
  • Ожидаемая дата закрытия

Что можно делать?

  • «Новая сделка» → Создать сделку с названием, email клиента, продуктом, суммой
  • Стрелки влево/вправо → Переместить в следующий/предыдущий этап
  • Редактировать (иконка карандаша)
  • Удалить (иконка корзины)
Совет: При перемещении сделки в «Проиграна» система спросит причину проигрыша.

8. Карточка клиента

/admin/kunden/[id]

360° обзор одного клиента. Две колонки:

Левая колонка — Профиль

  • Имя, email, телефон, дата рождения, язык, Telegram ID
  • Теги (цветные метки)
  • CRM-статус
  • Общая выручка, количество заказов и сессий

Правая колонка — История активности

Хронологический список всех взаимодействий:

  • CRM-заметки (заметка, звонок, email, follow-up)
  • Отправленные письма
  • Заказы
  • Сессии
  • Записи с формы контактов

Что можно делать?

  • Создать заметку: Выберите тип (Заметка/Звонок/Email/Follow-Up), напишите текст и опционально дату follow-up
  • Изменить статус: CRM-статус прямо на странице

12. Сессии (консультации)

/admin/sitzungen

Все консультации (бесплатные и платные).

Вкладки-фильтры

  • Все — Все сессии
  • Платные — Только оплаченные
  • Бесплатные — Бесплатные первые консультации

14. Контент и рассылки

/admin/content

Создавайте и отправляйте рассылки по email и Telegram.

Обзор

Список всех рассылок со статусом: Черновик, Запланировано, Отправляется, Отправлено, Ошибка.

Создание рассылки

  1. Заголовок и тема письма
  2. ИИ-ассистент (опционально): Выберите тип контента (Рассылка, Telegram-пост, Допродажа, Событие, Совет дня), язык и модель ИИ. Нажмите «Сгенерировать с ИИ»
  3. Редактирование: Вкладка «Email» для HTML, вкладка «Telegram» для текста
  4. Настройки:
    • Канал: Email, Telegram или оба
    • Аудитория: Все, VIP, Клиенты, Лиды или по тегам
    • Фильтр по языку: Все, только немецкий, только русский
  5. «Отправить сейчас» — Отправка с диалогом подтверждения (показывает кол-во получателей)
Совет: Количество получателей обновляется в реальном времени при изменении аудитории.

15. E-mail цепочки (автосерии)

/admin/sequenzen

Автоматические серии писем, которые запускаются при определённых событиях.

Как это работает?

  1. Вы создаёте цепочку с триггером (напр. «Лид создан» или «Заказ оплачен»)
  2. Добавляете шаги — каждый шаг это письмо с задержкой
  3. Когда триггер срабатывает, клиент автоматически подписывается и получает письма по порядку

Настройки шага

  • Задержка: Дни + часы после предыдущего шага
  • Тема и содержание письма
  • Telegram (опционально): Дополнительно отправить сообщение в Telegram

Доступные триггеры

Лид создан, лид подтверждён, профиль создан, заказ завершён, сессия завершена, тег добавлен, вручную.

Пример: Создайте цепочку «Добро пожаловать» с триггером «Лид создан»: Шаг 1 сразу (приветственное письмо), Шаг 2 через 3 дня (советы по нумерологии), Шаг 3 через 7 дней (предложение).

16. Теги и авто-теги

/admin/tags

Создайте правила, которые автоматически присваивают или удаляют теги на профилях клиентов.

Доступные условия

УсловиеПример
Заказов ≥ N«VIP» если ≥ 3 заказов
Выручка ≥ N«Премиум» если ≥ 500€
Купил продукт«PDF-покупатель» если купил PDF
Неактивен ≥ N дней«Неактивный» если ≥ 90 дней

Что можно делать?

  • Вкл/Выкл → Активировать или деактивировать правило
  • «Запустить сейчас» → Применить все правила ко всем профилям немедленно
Совет: Правила также запускаются автоматически каждую ночь в 3:00.

17. Автоматизации

/admin/automatisierung

Самая мощная функция CRM: правила «Если-То», которые автоматически выполняют действия.

Структура правила

  1. Триггер (ЕСЛИ): Событие, запускающее правило
  2. Условия (ТОЛЬКО ЕСЛИ): Опционально — дополнительные фильтры
  3. Действия (ТОГДА): Что должно произойти

Доступные триггеры

Лид создан, лид подтверждён, профиль обновлён, заказ завершён, возврат, сессия запланирована, сессия завершена, тег добавлен, тег удалён, форма контактов, CRM-статус изменён, follow-up, Instagram DM, Instagram лид.

Доступные действия

ДействиеЧто происходит
Добавить тегПрисваивает тег клиенту
Удалить тегУдаляет тег у клиента
Изменить статусМеняет CRM-статус
Отправить emailОтправляет письмо клиенту
Отправить TelegramОтправляет сообщение в Telegram
Создать заметкуСоздаёт CRM-заметку
Создать задачуСоздаёт задачу для вас
Подписать на цепочкуЗапускает email-цепочку
Отправить Instagram DMОтправляет личное сообщение в Instagram
Пример: ЕСЛИ заказ завершён → ТОЛЬКО ЕСЛИ пакет = pdf_analyse → ТОГДА добавить тег «PDF-покупатель» + отправить email «Спасибо» + создать задачу «Follow-up через 7 дней».

Журнал выполнения

Во вкладке «Выполнения» видны последние 30 запусков со статусом (Успех/Частично/Ошибка).

24. Купоны

/admin/gutscheine

Создавайте и управляйте промокодами для магазина.

Поля при создании

  • Код: Промокод (автоматически в заглавных буквах)
  • Тип: Процент (%) или фиксированная сумма (€)
  • Значение: напр. 15 (для 15% или 15€)
  • Макс. использований: Пусто = безлимитно
  • Действует с/до: Период действия
  • Применяется к: Все продукты / Только PDF-анализ / Только консультации
  • Stripe ID (опционально): Для интеграции с Stripe Checkout

Что можно делать?

  • «Новый купон» → Создать
  • Редактировать (иконка карандаша)
  • Деактивировать (иконка корзины)
  • Скопировать код (иконка копирования)
